# React Summary

## 1. Introduction
React est une bibliothèque JavaScript open‑source développée par Facebook (maintenant Meta) pour construire des interfaces utilisateur interactives.  
Il se concentre sur la composition d’**composants** réutilisables, l’**unidirectionnalité** des flux de données et une **performance** optimisée grâce au DOM virtuel.

## 2. Concepts clés
| Concept | Description |
|---------|-------------|
| **Composants** | Fonctions ou classes qui retournent du JSX (ou null). Ils encapsulent logique et rendu. |
| **Props** | Paramètres en lecture seule passés aux composants enfants. |
